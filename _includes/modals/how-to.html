How to Use this Site

There are a few conventions used on this site to make navigating through it quicker and easier.

  • opens a tutorial module card that will give step by step instructions. Clicking on the button again closes the module.
  • A link like this opens a card that explains a key term. There are likely tutorial modules embedded in that explanation. Clicking on the link also closes the card.
  • | signals that the following content is operating system–specific. Clicking on the icon appropriate to your system tailors the instructions appropriately. The darker icon is the currently active one.
  • Keys like this signal keyboard commands to type, all at once. On a Mac, cmd is the key labeled “command” and “⌘.”
  • Text like this, however, is text to be typed in, not all at once. It can also refer to file names.

  • Sometimes tutorials include text that

    {% highlight shell linenos %} looks like this {% endhighlight %} This indicates either a series of commands or commands to type into the terminal.

Hopefully this is sufficient to lead you to plain text success.
